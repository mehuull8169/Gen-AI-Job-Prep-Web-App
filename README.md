# 🚀 Gen AI Job Prep Web App

A full-stack web application that helps users prepare for job interviews using AI-powered mock interviews and feedback.

---

## 📌 Overview

This project simulates real interview scenarios where users can practice answering questions and receive AI-generated feedback. It is designed to improve confidence, communication, and interview performance.

---

## ✨ Features

- 🎤 AI-based mock interview system  
- 📊 Real-time feedback and evaluation  
- 🔐 User authentication (Login/Register)  
- 🧠 Dynamic question generation  
- 📄 Interview report generation
